# Coal Shipment Operational Analysis Dashboard

A comprehensive Power BI solution designed to monitor and optimize coal shipment operations. This dashboard provides end-to-end visibility into logistics performance, weather impacts, and supply chain risks, specifically tailored for analysts and sales teams.

## Dashboard Modules
The dashboard is structured into four key analytical areas:
1. **Executive Overview:** High-level tracking of shipment KPIs and operational efficiency.
2. **Weather Analysis:** Correlation of meteorological conditions with shipment schedules to mitigate delays.
3. **Coal Analysis:** Monitoring coal quality parameters and inventory levels.
4. **Shipment & Risk Analysis:** Leveraging **Python-based algorithms** to identify bottlenecks and predict logistical risks.

## Key Features & Logic
- **Advanced DAX Measures:** Implementation of complex time-intelligence and dynamic ranking for shipment performance.
- **Python Integration:** Automated risk scoring and user behavior analysis using Python scripts within the Power BI environment.
- **Data-Driven Pitching:** Designed with insights ready for pitching to stakeholders, focusing on cost reduction and operational reliability.

## Tech Stack
- **Power BI:** Primary tool for data visualization and modeling.
- **DAX:** For advanced calculated measures and KPI logic.
- **Python:** Used for predictive risk algorithms and advanced data processing.
- **Power Query (M):** ETL processes for cleaning and transforming coal shipment logs.
